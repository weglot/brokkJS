## How to Use

### Via jquery plugin

```javascript
$(document).ready(function(){
    $(selector).wejax({});
});
```

### Via data attribute

```html
<div data-url></div>
```

```javascript
$(document).ready(function(){
    $(selector).wejax();
});
```

#### Any option can be passed through the use of a data attribute or plugin object.
